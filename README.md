# Ejecutar en desarrollo

1. Clonar repo
2. Ejecutar

```
yarn install
```

3. Tener Nest CLI instalado

```
 npm i -g @nest/cli
```

4. Levantar la base de datos

```
docker-compose up -d
```

## Stack usado

* MongoDB
* Nest
